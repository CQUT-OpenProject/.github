### 关于我们

「重庆理工大学开源计划」（本组织）是致力于**收集和维护本校开源项目**的非官方、非盈利组织，旨在促进资源分享与推广开源社区文化。

由于 [课程攻略共享计划](https://github.com/Royfor12/CQUT-Course-Guide-Sharing-Scheme) 为单体仓库，不利于代码类资源的管理。于是我们创建了本组织，将《共享计划》中的部分内容，迁移并分散至本组织的不同的仓库中进行维护。

### 加入我们

在 [Issue](https://github.com/CQUT-OpenProject/.github/issues/1) 中提交任意评论以加入组织，加入后可获得仓库管理权限。

为防止权限滥用，请求由管理员人工审核。审核通过后，请于 Github 关联邮箱内接受邀请链接。

## 贡献 ｜ 上传资源

本组织有以下两种贡献方式：

1. [加入组织](###加入我们)后直接管理
2. 在此仓库[新建 Issue](https://github.com/CQUT-OpenProject/.github/issues)，作为附件上传资源，由项目维护者进行添加

☁️ 建议上传的资源类型：

- 平时作业、课程设计
- 需要长期更新的项目
- 需要单独管理的大型项目
- 服务于、或与学校相关的开源软件/项目

❓ 不建议上传的资源类型：

- PPT、笔记、期末试卷、电子书、物理实验报告

> 此类内容可另上传至《共享计划》
