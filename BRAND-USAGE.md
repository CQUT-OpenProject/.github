# CQUT-OSP 品牌使用政策

品牌使用政策适用于“CQUT-OSP”、“重庆理工大学开源计划”、CQUT 四字母模块化标志及其官方组合。相关条款不影响代码和文档已采用的开源许可证。

## 允许使用

- 真实引用、介绍或评论 CQUT-OSP 及其项目。
- 在贡献者简历、文章或演示中准确说明参与关系。
- 链接到组织官方 GitHub 页面或使用未修改标志指向组织资源。
- 经维护者批准，为社区活动制作符合品牌规范的宣传材料。

## 必须满足

- 不得暗示重庆理工大学、院系或行政机构的官方背书。
- 不得暗示第三方项目、账号或服务由 CQUT-OSP 官方运营。
- 标志必须遵守安全区、颜色、最小尺寸和背景规则。
- 开展加入组织、下载资源或身份验证等活动时，严禁以社区名义收费或误导用户。

## 禁止使用

- 冒充组织或学校官方账号、登录页、认证服务、通知或证明。
- 将名称和标志用于诈骗、恶意软件、隐私侵害或其他误导性活动。
- 注册足以造成混淆的域名、组织名、应用名或社交账号。
- 修改标志后继续声称其为 CQUT-OSP 官方标志。
- 未经许可将品牌用于商品销售、付费服务或商业背书。

## 第三方与学校标志

本政策不授权使用重庆理工大学校徽、名称标志或其他第三方商标。

## 联系与授权

若需在上述授权范围之外使用品牌资产，请通过 [CQUT-OpenProject GitHub 组织](https://github.com/CQUT-OpenProject)的公开渠道联系维护者。对于可能造成混淆或损害社区形象的使用行为，维护者有权要求予以中止。
