## 标准非官方声明

> 重庆理工大学开源计划是由学生及贡献者维护的非官方开源社区，不代表重庆理工大学或任何校内机构。

网页页脚、关于页面以及首度触及校内身份的业务流程中必须展示完整声明。在空间受限的界面中，可采用简短形式“非官方社区项目”，并附带指向完整说明的链接。

## 高风险场景

在涉及登录、统一身份认证、校园卡、课表、成绩、缴费或个人隐私信息的页面上，需另外公开以下信息：服务运营主体、所读取的字段、数据存储方案、销毁机制、与学校官方系统的对应关系，以及故障反馈渠道。

不得使用“学校认证”“官方授权”等无法证明的表述。`CQUT-Auth` 等身份项目只能说明验证了什么技术事实，不能把社区账号包装成学校官方凭证。

## 来源署名

转载或改编的内容需注明原作者或项目名称、原始链接、授权许可证以及是否经过修改。代码仓库中引入的第三方代码与资产继续遵循其原始许可证，不因并入社区项目而自动变更授权协议。

## 防诈骗表达

涉及加入团队或获取相关资源时，需明确提示“重庆理工大学开源计划为非营利社区，严禁任何人以加入、下载或审核为名目收取费用”。若发现有账号冒用社区品牌，请保留相关证据并向 GitHub 组织公开渠道进行核实。
